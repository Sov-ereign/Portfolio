Bulls and Cows Solver

An interactive solver for the classic game Bulls and Cows, helping users find optimal guesses.

Designed a web-based solver that assists players in cracking the Bulls and Cows code efficiently. But unlike the traditional bulls and cows this solver is capabale of solving codes beyond the length of 4 difgits only.

Well for now we are going with 4 digits and the number of possibilities here are 0-9 = 10 numbers so -- 10!/4!*6! =210, 210 * 4! = 210 * 24 = 5040 possibilities

Implements an algorithm to analyze user input and suggest the next best guess. Going with that recommended next guess will reduce the number of possibilities in the next move.

Displays real-time feedback on guesses, helping users refine their approach.
